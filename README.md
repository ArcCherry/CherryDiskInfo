**Android 存储健康检测工具**

CrystalDiskInfo 的 Android 精神续作，为无法更换存储的移动设备提供寿命透明度。

### 为什么做这个项目

PC 用户有 CrystalDiskInfo 监控 SSD 寿命，但手机/平板用户面对焊死的 eMMC/UFS 却一无所知。市场充斥着存储濒临报废的设备，而 Android 直到 15 才提供官方 API，旧设备更是黑箱
